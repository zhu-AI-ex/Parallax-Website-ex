視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動 (Parallax scrolling) 是一種網頁設計技術，其中網站背景的移動速度比前景慢。這會在訪客向下滾動網站時產生 3D 效果，增加深度感並創造更身臨其境的瀏覽體驗。視差是基於光學錯覺。 由於人眼認為靠近我們的物體比遠處的物體大，我們會認為遠處的物體移動得較慢。這種錯覺早已被應用於各種媒介的視差中，以產生逼真的效果。 它最早用於傳統動畫，可追溯到迪士尼的《白雪公主和七個小矮人》，以及《超級瑪利歐》等電玩遊戲。 隨著 CSS 和 HTML 的進步，視差效果後來演變成了我們今天所知的網頁設計領域。

瀏覽器中最早的視差滾動實作之一是由 Web 開發人員 Glutnix 在 2007 年的一篇部落格文章中創建和分享的，其中包括使用 JavaScript 和 CSS 2 的範例程式碼和演示， 支援 Internet Explorer 6 和那個時代的其他瀏覽器。然而，直到 2011 年 HTML5 和 CSS 3 編碼的引入，該技術才在網頁設計師中流行起來。 倡導者認為這是擁抱網路流動性的簡單方法。支持者使用視差背景作為工具來更好地吸引用戶並改善網站提供的整體體驗。 然而，普渡大學 2013 年發表的一項研究揭示了以下發現：「...雖然視差滾動增強了用戶體驗的某些方面，但它並不一定能改善整體的用戶體驗」。


為什麼要使用視差圖形？

    設計師應該嘗試視差圖形的主要原因有兩個：

  • 它有助於講述故事

    視差圖形非常適合講故事。好的設計在於溝通——當我們設計網站時，我們希望與訪客溝通，講述關於我們的產品、服務或品牌的引人入勝的故事。 視差滾動使我們能夠讓網站訪客沉浸在獨特的線上體驗中。甚至可以創建一個單頁網站，讓訪客閱讀關於您的產品或品牌的完整故事，而無需導航到多個頁面。

  • 它有助於提高使用者參與度

    視差圖形可用於提高使用者參與度。降低網站的跳出率 (Bounce rate) 是網頁設計中需要認真對待的目標之一。 跳出率衡量的是只查看了您網站的一個頁面就點擊「返回」按鈕或關閉分頁的訪客百分比。 高跳出率會損害您的搜尋排名，因此任何能降低該比率的行動都是值得的。精心製作的視差頁面能為使用者提供一種動態體驗，往往能讓他們在頁面上停留更長時間。


如果您想在網站中添加視差滾動效果，請考慮以下建議：

  • 選擇性地使用

    當您將視差滾動用於特定元素而不是所有頁面和所有時間時，它的功能要強大得多。將其用於頁眉和標題，或者可能只用於您的主頁。 您希望視差滾動能增強您的網站設計，而不是分散使用者的注意力。

  • 圖像壓縮

    視差滾動使用各種媒體檔案和 CSS 形狀來創造深度感。您不希望因為加載過慢的大型圖像檔案而損害您的網頁效能和使用者體驗。

  • 玩轉色彩

    視差滾動不僅僅是關於前景和背景中的圖像，還關於顏色。您可以在背景中使用更不飽和的色調，在前景中使用更生動的顏色，以創造更強的深度感。
